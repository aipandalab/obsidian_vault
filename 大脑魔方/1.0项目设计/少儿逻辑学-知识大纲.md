# 1 认知发展适配与建构

## 1.1 具象化逻辑建构
### 1.1.1 适配年级：1-2年级

该大纲通过"具象操作→符号过渡→关系网络"的渐进路径，将逻辑学核心概念转化为可触摸、可互动的学习体验。建议配套开发《熊猫逻辑小川侠》系列动画（每集约8分钟），用四川方言解说逻辑原理，同时建立家校联动的"家庭逻辑角"，定期发布如"给熊猫设计最佳进食路线"等亲子挑战任务。

从儿童认知发展理论出发，小学一年级（6-7岁）学生正处于皮亚杰认知发展理论中的**前运算阶段**向**具体运算阶段**过渡期，符号化概念的启蒙教学需要遵循"具象→半抽象→符号"的认知路径。

### 1.1.2 知识清单
	实体关系认知
		简单命题、主词、谓词
		
		逻辑思维点：多维度特征提取、理解分组的基本逻辑
		知识点：单属性分类、多属性交集分类、动态条件分类
	排序、序列
	逻辑连接词（与、或、非）
	
	符号化：实物、图片、文字、符号
	
	分类：基于“演绎推理”，按既定规则归类，逻辑结构：直言三段论，对应：监督学习
	聚类：基于“归纳推理”，先发现数据内在结构再归类，逻辑结构：特殊到一般，对应：无监督学习
	- **分类**​：  
    `已知类别标签 → 学习区分规则 → 应用规则判断`  
    （如：先认识"水果/蔬菜"定义，再判断番茄类别）
    - ​**聚类**​：  
    `观察数据特征 → 发现相似模式(模式识别) → 定义分组标准`  
    （如：先观察番茄、苹果、胡萝卜的特征，后自主分组）
    
	集合、比较、克隆、比例、归纳
	边界条件的敏感性
	条件关系、条件推理(先判断，后行动，条件清楚不碰钉)
	同一关系：同一律
	排中关系：排中律，命题真值
	互斥关系：矛盾律
	序列预测
	简单命题演算：（与、或、非）
	CPA问题建模
### 1.1.3 工具推荐
	思维导图、乐高积木、scratch
### 1.1.4 能力目标
	概念理解：情景问答、举例说明
	建立事物分类的元认知    AI衔接点：机器学习中的监督学习数据标注基础
	理解事件顺序的不可逆性    AI衔接点：算法执行流程的时序控制
	识别互斥关系的基本形态
	思维模式(结构化思维)：
		拆解策略：拆解问题的完整性
			将制作担担面分解成多个步骤、班级大扫除任务拆解成小组分工
		模式识别：找规律，川菜的特点，古诗，建筑特色
		调试思维：机器人走迷宫指令修正
### 1.1.5 案例推荐

	魔法厨房
	三星堆文物修复排序

	使用四川特色元素（如大熊猫食物分类：竹子/竹笋/水果）
	嵌套分类游戏（三星堆文物按材质→用途→年代分层归类）
	川剧变脸角色出场顺序重构
	泡茶的流程
	都江堰水利工程运作流程拼图
	同一张脸谱不可能同时是红脸和黑脸
	机器人不能同时执行前进和后退指令
