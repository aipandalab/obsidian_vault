**3. 数学谜题：阶乘计算**

- **挑战描述：** 计算一个非负整数的阶乘。
- **解题思路：** 读取一个非负整数 n。如果 n 为 0，阶乘为 1。否则，阶乘是 1 * 2 * ... * n。可以使用循环来计算。
- **参考答案 (Python):**
    
    Python
    
    ```Python
    n = int(input())
    factorial = 1
    if n == 0:
        factorial = 1
    else:

            factorial *= i
    print(factorial)
    ```

- **参考答案 (Python):**
    
    Python

    ```Python
    word = input()
    k = int(input())
    encrypted_word = ""
    
    for char in word:
        if 'a' <= char <= 'z':
            start = ord('a')
            shifted_char_code = start + (ord(char) - start + k) % 26
            encrypted_word += chr(shifted_char_code)
        else:
            encrypted_word += char # Should not happen based on problem description
    print(encrypted_word)
    ```

- **解题思路：** 读取N和N个人名，将人名存入一个列表（模拟队列）。读取M，然后从列表的开头移除M个元素。最后输出列表中剩余的元素。
- **参考答案 (Python):**
    
    Python
    
    ```Python
    n = int(input())
    queue = []
    for _ in range(n):
        queue.append(input())
    
    m = int(input())
    
    # Dequeue M people
    dequeued_count = 0
    while dequeued_count < m and queue:
        queue.pop(0)
        dequeued_count += 1
    
    for person in queue:
        print(person)
    ```
    
- **针对样例输入 的输出:**
    
    ```
    Charlie
    David
    Eve
    ```
