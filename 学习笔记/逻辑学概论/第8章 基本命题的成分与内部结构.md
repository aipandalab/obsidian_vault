# 1 基本命题的结构
	命题：对于事物(主词)及情况(性质、关系)(谓词)的陈述
		所有  金属  是导体
		有些   人   会游泳
	基本命题组成部分：主词(S)、谓词(P)、量词、联词(主词与谓词之间肯定或否定)
	主词和谓词统称词项——事物、事物的情况(性质或关系)

# 2 词项的内涵和外延
	内涵：某一词项的含义，即该词项所指对象共同具有的特有属性
	外延：某一词项所指的对象。比如金属的外延是：金、银、铜、铁……
	内涵和外延之间有反变关系
		增加词项的内涵以缩小外延
		减少词项的内涵以扩大外延
	学校：专门从事教育的机构
	大学：专门从事高等教育的机构

# 3 词项的种类
	根据词项外延的数量情况，词项分为：
	普遍词项：外延超过一个；比如：学校
	单独词项：外延只有一个；比如：清华大学
	空词项：外延为空；比如：美国女总统(内涵很清楚，但现实外延至今没有)

# 4 词项间的关系
	词项之间的关系：指词项外延之间的关系。
	(1)全同(同一)关系
 ![image.png|196](https://panda-ai-316.oss-cn-chengdu.aliyuncs.com/obsidian/202505022027350.png)
	比如：中国的首都与华北最大的城市是全同(同一)关系
	(2)包含关系
	![image.png|205](https://panda-ai-316.oss-cn-chengdu.aliyuncs.com/obsidian/202505022034335.png)
	比如：中国的学校S与中国的大学P之间就是包含关系
	(3)包含于关系
	比如：中国的大学P与中国的学校之间就是包含于关系
	(4)交叉关系
	![image.png|203](https://panda-ai-316.oss-cn-chengdu.aliyuncs.com/obsidian/202505022058283.png)

	(5)全异关系
 ![image.png|302](https://panda-ai-316.oss-cn-chengdu.aliyuncs.com/obsidian/202505022100439.png)
	 分为：
		 矛盾关系：没有共同外延，但加起来是更高一级的词项，即互补
		 比如：一个学校中的男生和女生，加起来是该校的全体学生
		 反对关系：不互补，比如：大学生和小学生之间存在其他类型学生

# 5 词项的定义
	定义：描述词项的内涵
	定义的结构：被定义项，定义项
		例如：偶数  能被2整除的整数
	同一个词项可有不同的定义，比如：水的物理定义和化学定义
	- 定义的主要规则：
	(1)定义项和被定义项必须为全同关系
		定义过宽：鱼是生活在水中的动物
		定义过窄：期刊是每月定期出版的出版物
	(2)定义项不得直接或间接包含被定义项
		同语反复：
			逻辑学是研究逻辑的学问
		循环定义：
			偶数是奇数加1或减1得到的数
			奇数时偶数加1或减1得到的数
			避免循环定义，必须要有定义出发点(初始符号、初始定义)
	定义不是获得知识的唯一方法，而是来自实践生活，比如：上下左右

# 6 词项的划分
	词项的划分：分类列举词项的外延
	划分的结构：母项、子项
	生物分为：动物、植物、微生物
	(母项)  (      子项     )
	同一个词项可以按不同标准做不同的划分
	划分可连续进行，即：子项可作为母项再次进行划分
	
	划分的主要规则：
	(1)一次划分必须按同一标准进行
	(2)每一外延应属于某一子项并只属于一个子项(户籍)。即：子项相加应恰等于母项，不得遗漏(其他选项)；子项之间应相互排斥，不得重合。(一块饼分隔)

# 7 谓词的分类
	谓词影响命题类型：
		例1：张三和李四是高中学生——合取关系的复合命题
		例2：张三和李四是同班同学——基本命题
	一元谓词：每次需要一个主词与之配合，通常表示主词的某种性质；如上例1
	多元谓词：每次需要多个(两个以上)主词与之配合，通常表示多个主词之间的某种关系；如上例2。
	性质命题：含有一元谓词的基本命题
	关系命题：含有多元谓词的基本命题
		例3：5大于3；5是一个主词，3是一个主词，两个主词之间是大于关系
		例4：西安在兰州和郑州之间，在……之间是谓词，含三个主词

# 8 量词
	量词用于限定外延的范围，分为：
	全称：所有的、都、全部……，表示主词所有外延都适用
	特称：(存在)有的、有些……，至少存在一个外延适用(不排斥全部)
	单称：可以表示独一无二的一类东西，也可以表示某一个具体的外延，通常处理为全称。比如：一人吃饱全家不饿
	全称量词可省略。比如：金属是导体
