## 为什么这次更新如此重要？

[00:33在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=33) 首先，让我们看看为什么这次更新是个大事件。简单来说：

- **Hermes** 是由 Nous Research 构建的开源 AI Agent。它不仅仅是一个聊天机器人，它能为你执行任务、使用工具、搜索网页、写入文件、运行命令，并且会随着你的使用进行学习。
    
- **核心优势**：它能记住你做过的事情并自己构建小技能。你用得越多，它就越擅长处理你重复要求它做的事情。现在，它推出了官方桌面应用（Desktop App），你再也不用挤在终端（Terminal）里使用它了。
    
- **Ollama**：这是一个免费工具，可以让你在自己的电脑上直接下载并运行 AI 模型。无需账号、数据绝不离开本地机器。
    

[01:06在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=66) 很多人忽略了一点：Ollama 增加了一个名为 `ollama launch` 的命令，可以为你自动配置你最喜欢的 Agent 和编码工具。Hermes 就是其中之一。因此，你不需要折腾配置文件，也不需要复制粘贴一堆设置。你只需运行一条命令，它就会引导你完成接下来的步骤。

## 一键配置步骤

[01:21在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=81) 让我来展示这个配置有多简单：

1. 前往 `ollama.com` 下载软件，它支持 Mac、Windows 和 Linux。如果你之前用过，请确保更新到最新版本。
    
2. 打开终端（Terminal），输入：
    
    Bash
    
    ```
    ollama launch hermes
    ```
    
3. [01:32在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=92) 就这样！这一条命令会检查是否安装了 Hermes，如果没有，它会提示使用 Nous Research 的官方脚本为你安装。
    
4. 接着你可以选择一个模型（无论是运行在本地的还是云端的）。它会自动将 Hermes 指向你的本地 Ollama 地址。
    
5. 它甚至还能连接 Telegram、Discord、Slack 或 WhatsApp 等通讯软件，这样你就可以通过手机与你的 Agent 对话了。
    

[01:53在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=113) 我在 AI Profit Boardroom 的研究中就使用了这套完全相同的配置，同时运行了一批研究 Agent。一个 Agent 负责收集新成员不断提出的问题，另一个负责起草简短的演示大纲来回答这些问题，第三个则按主题对所有内容进行分类。它们同时运行，每个都有自己干净的上下文，不会产生任何混淆。

## 为什么选择桌面版（Desktop App）而非终端？

[02:19在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=139) 你可能会问，为什么要用 Hermes Desktop？以前在终端里运行不是挺好的吗？

坦白说，终端虽然能用，但很痛苦。在终端输入 `hermes`，除了滚动的文本你什么也看不到。你无法预览工作、无法轻松回顾过去的聊天，没有一个好地方来管理任何东西。

[02:37在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=157) 桌面应用解决了所有这些问题，所有东西都在一个地方：

- 你可以看到你的技能、消息、文件、图像以及保存的链接。
    
- 你可以随时切换到设置，更改外观、设置聊天，或者在任务中途点击几下鼠标来**更换正在使用的模型**。
    
- （个人建议：开启暗黑模式，长时间工作时对眼睛更好。）
    

[03:00在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=180) 在设置中还有很多神奇的功能：你可以选择你的模型供应商、开启或关闭技能、配置你的消息通道并更改应用的外观。

> ⚠️ **关键提示**：Hermes 需要一个**至少有 64,000（64K）上下文 Token** 的模型才能正常工作。这是它处理多步骤任务所需的“工作内存”。大多数云端模型很容易达到这个标准，但如果你运行本地模型，**你必须自己设置上下文大小**，否则 Hermes 会在启动时拒绝它。

## 推荐的免费本地模型

[04:06在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=246) 接下来聊聊我承诺过的免费本地模型。通过 Ollama 在本地运行模型是真正免费的，唯一的成本就是你的电费，没有 Key、没有限制、没有封顶。

如果你有一台配置不错的机器，我推荐以下几个：

- **Qwen 2.5**：非常强大的选择，擅长推理、代码，甚至能理解图像。
    
- **GLM 4**：另一个非常扎实的选择。
    
- **Minimax-Text-01 (M3)**：最新推出，这是一个真正前沿级别的开源模型，拥有惊人的 **100万（1M）上下文窗口**，非常适合处理长期复杂的任务。
    

[04:38在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=278) 但坦白讲：模型越大、越聪明，对电脑配置的要求就越高。像 Minimax 或英伟达的 Nemotron-3 Ultra 固然不可思议，但它们需要过硬的硬件才能在本地跑得顺畅。

如果你的设备只是普通的笔记本电脑，也不用担心。你可以运行更轻量级的模型，比如 **Gemma 2**。较小版本的 Gemma 2 可以在 16GB 内存的笔记本上运行，而且效果非常不错。你只需要权衡利弊：配置越高，模型越大，结果越好；配置轻量，模型较小，但依然非常有用。

## 核心高级功能

### 1. 并行 Agent（Parallel Agents）

[05:10在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=310) 这是我最喜欢的功能之一。Hermes 可以衍生出所谓的“子 Agent”（Sub-agents），每个子 Agent 都有自己独立的上下文。简单来说，你可以并排运行多个 Agent，它们之间不会产生记忆污染。一个专注于研究，一个专注于写作，一个专注于分类，各自保持干净的记忆。你可以在应用底部轻松切换。

### 2. 本地模型的四大理由

1. **完全免费**
    
2. **隐私安全**（数据不离开电脑）
    
3. **离线工作**（在飞机上没网也能用）
    
4. **纯粹的乐趣**（在自己的机器上运行智能 AI 真的很酷）
    

### 3. 全平台同步

[06:22在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=382) 你的 Hermes Agent 在不同应用和通道（如 Telegram）上的对话，都会同步显示在桌面应用里。你在手机 Telegram 上聊的内容，回家打开桌面版都在那里。

### 4. 自动化任务计划

[06:42在新窗口中打开](http://www.youtube.com/watch?v=6Rg6UvFNvf4&t=402) Hermes 内置了任务计划程序，你用简单的英语就能设置它。例如：“每天早上运行某个任务”、“每晚备份”或“每周检查一次”，它就会在后台自动处理。即使应用关闭了，它也会继续运行，并将结果直接发送到你的 Telegram 或 Discord。
