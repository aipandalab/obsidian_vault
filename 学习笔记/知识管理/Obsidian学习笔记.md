# 4 双链

包含入链与出链，任何一个从A到B的链接，站在A的角度看该链接就是出链，站在B的角度看该链接就是入链。

```
创建链接：

链接到某篇文章：两个左中括号+文章标题+两个右中括号
链接到某个段落：两个左中括号+文章标题#段落标题+两个右中括号
链接到文本块：两个左中括号+文章标题^文本块编号+两个右中括号
链接别名：两个左中括号+文章标题^文本块编号｜链接别名+两个右中括号
显示连接内容：![[学习笔记#^26196d|基本语法]]
预览链接内容：按住command键，移动鼠标到链接上
```
